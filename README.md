# Fon - Focus On Now

Fon 是一个自用的待办事项(Todo List)应用程序。

## ✨ 特性

- 🗄️ **本地存储** - 基于 SQLite 的本地数据库，安全可靠
- 🗂️ **分类视图** - 所有任务、标签分组、已删除等多视图管理
- 📝 **富文本编辑** - 基于 Tiptap 的强大内容编辑器，支持 Markdown
- 📎 **文件管理** - 支持图片和文件附件，拖拽上传
- ✅ **任务列表** - 支持任务清单、代码块、表格等多种内容格式

## 文件存储

- 数据存储在本地 SQLite 数据库,位于程序运行目录下的 `data/db` 文件夹
- 附件位于程序运行目录下的 `resources` 文件夹
